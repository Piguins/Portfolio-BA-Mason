## Quick Start

```bash
# Clone repository
git clone https://github.com/Piguins/Portfolio-BA-Mason.git
cd Portfolio-BA-Mason

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

- React 18.2.0
- Vite 5.0.8
- React Icons 4.12.0
- Plain CSS with CSS Variables
